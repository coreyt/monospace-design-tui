This companion document to the [Design Standard](/standard/) provides the exact character codes, escape sequences, color mappings, and component measurements needed to implement compliant Monospace TUI applications. Where the standard says *what* to do, this reference says *how* — down to the Unicode codepoint and SGR parameter.
